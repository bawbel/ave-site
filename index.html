The behavioral classification standard for agentic AI

Stable IDs, AIVSS scores, and behavioral fingerprints for every way an agentic component — skill file, MCP server, or agent plugin — can be weaponized. Scored consistently. Mapped to OWASP MCP Top 10, MITRE ATLAS, and the frameworks security teams already report against.

AVE is an open standard. The schema and records are published under Apache 2.0. AVE is moving toward neutral governance under an independent standards body — it is meant to outlive any single implementation, including Bawbel's.

Maps to OWASP MCP Top 10 · OWASP Agentic AI Top 10 · MITRE ATLAS · OWASP AIVSS v0.8. The open record set can be consumed by any tool.

Citing AVE. Reference a record by its id, which is permanent and never reused: AVE-YYYY-NNNNN: Title. Agentic Vulnerability Enumeration (AVE), Bawbel/AVE Project. https://ave.bawbel.io/registry.html#AVE-YYYY-NNNNN
